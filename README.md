# EJECT-Cumpump
Eject Cumpump which works together with the OSSM, controlled by M5 Remote
This project is the firmware for the EJECT-Cumpump, a device designed to work in conjunction with other repositories in the EJECT system. It is forked from [ortlof/EJECT-cum-tube-project](https://github.com/ortlof/EJECT-cum-tube-project) and adapted for improved modularity and integration.

## Overview

The EJECT-Cumpump controls the pump mechanism, providing precise movement and control for the EJECT system. You can control the cumpump using https://github.com/MarcoB979/OSSM-M5-Remote-with-Eject-and-Fist-IT

## Integration

This firmware is designed to work together with the M5 remote and OSSM. Please refer to the documentation in those repositories for integration instructions.
